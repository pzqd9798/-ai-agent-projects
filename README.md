## 技能体系

- **Agent 框架**: LangGraph, AutoGen, Camel, AgentScope, HelloAgents
- **后端**: FastAPI, Flask, Python, TypeScript
- **前端**: Vue3, Vite, Streamlit, Ant Design Vue
- **LLM**: OpenAI, Anthropic Claude, DeepSeek, Ollama
- **Agent 核心技术**: RAG, MCP, A2A, Prompt Engineering, Tool Use, Memory
- **强化学习**: SFT → GRPO 训练流程
- **基础设施**: Docker, Redis, Git, Linux
