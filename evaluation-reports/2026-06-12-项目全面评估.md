## 第二梯队（辅助项目，25-33 分）

### 4. AutoGen_software_team — 33 分

- AutoGen 框架实战，4 Agent 软件研发流水线
- 前后端分离：FastAPI + SSE 流式 + Streamlit 角色颜色展示
- 3 个预设任务 + 自定义任务支持
- **价值**: 框架多样性证明（四大框架全了）

### 5. rag-agent — 30 分

- 从零实现完整 RAG 管道（摄取→分块→向量化→检索→增强→生成）
- TF-IDF 向量化器零外部依赖，支持中文 2-gram 分词
- 双记忆系统（短期对话+长期偏好）
- **价值**: 展示底层理解，不是调包侠

### 6. job-agent — 28 分

- browser-use 自动化搜索 + LLM 7 维度简历匹配
- 双模式设计：演示模式（规则引擎零依赖）+ 真实浏览器模式
- **价值**: 浏览器自动化 + 实用性强

### 7. web-agent — 27 分

- 双模网页提取（静态HTTP+browser-use）+ LLM 结构化摘要
- 批量并发处理
- **价值**: 体现工程能力（并发、多格式输出）

### 8. daily-digest — 25 分

- 多源聚合（HN、GitHub、知乎、RSS）+ 定时调度 + Telegram 推送
- **价值**: 展示全自动化 pipeline 能力

## 面试策略建议

### 必须准备的故事线

**"从原理到产品"叙事**：
> "我先学了 Agent 底层原理（claw0），然后分别用 LangGraph 和 AutoGen 做了实战项目，最后自己从零设计开发了一个 Agent 平台 ClawBot，带管理后台和 Docker 部署。全链路都能讲。"

### 针对不同岗位的话术

| 岗位 | 主打项目 | 关键话术 |
|------|---------|---------|
| AI Agent 开发 | ClawBot + trip-planner | "手写核心循环，不依赖LangChain；MCP协议集成实战" |
| Python 后端 | ClawBot + deepresearch | "FastAPI SSE流式、SQLite多租户、Redis双记忆" |
| 全栈开发 | trip-planner + deepresearch | "FastAPI+Vue3全栈，Pydantic强类型Schema" |
| AI 产品 | ClawBot | "从0到1的产品化经验，有管理后台和场景模板" |

### 弱项诚实指出

- 无分布式/高并发生产经验
- 无 CI/CD 自动化流程
- 无云服务部署经验（AWS/阿里云）
- 测试覆盖率低
